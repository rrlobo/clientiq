Setup — First time configuration

You need two things: a Google OAuth Client ID and an Anthropic API key. These are saved locally in your browser only.

1 Go to console.cloud.google.com → APIs & Services → Credentials → Create OAuth 2.0 Client ID. Set type to Web application and add your file path or http://localhost to authorized origins.
2 Enable the Google Sheets API in your Google Cloud project under APIs & Services → Library.
3 Get your Anthropic API key from console.anthropic.com.
